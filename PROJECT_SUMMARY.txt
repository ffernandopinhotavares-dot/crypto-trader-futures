TABELAS PRINCIPAIS

1. users
   - id (PK)
   - email
   - username
   - passwordHash

2. bybitApiKeys
   - id (PK)
   - userId (FK)
   - apiKey
   - apiSecret
   - isActive
   - testnet

3. tradingConfigs
   - id (PK)
   - userId (FK)
   - name
   - tradingPairs (JSON)
   - maxPositionSize
   - stopLossPercent
   - takeProfitPercent
   - Indicadores (RSI, MACD, BB, EMA)
   - timeframe

4. trades
   - id (PK)
   - userId (FK)
   - configId (FK)
   - symbol
   - side (BUY/SELL)
   - entryPrice
   - exitPrice
   - quantity
   - pnl
   - status (OPEN/CLOSED)
   - stopLoss
   - takeProfit

5. candles
   - id (PK)
   - symbol
   - timeframe
   - timestamp
   - open, high, low, close
   - volume

6. indicators
   - id (PK)
   - symbol
   - timeframe
   - rsi, macdLine, bbUpper, ema
   - volumeMA

7. tradingLogs
   - id (PK)
   - userId (FK)
   - logType
   - message
   - details (JSON)

8. botStatus
   - id (PK)
   - userId (FK)
   - isRunning
   - totalTrades
   - winningTrades
   - totalPnl
   - winRate

================================================================================
                              COMO USAR O SISTEMA
================================================================================

1. INSTALAÇÃO
   $ pnpm install
   $ pnpm run db:push

2. CONFIGURAÇÃO
   - Editar .env com credenciais MySQL
   - Obter chaves Bybit API
   - Configurar na interface web

3. EXECUÇÃO
   Desenvolvimento:  $ pnpm run dev
   Produção:        $ pnpm run build && pnpm run start

4. ACESSO
   Desenvolvimento: http://localhost:5173
   Produção:       http://localhost:3000

5. OPERAÇÃO
   - Ir para "Configuração" > "Nova Configuração"
   - Preencher parâmetros da estratégia
   - Ir para "Dashboard" > "Iniciar Bot"
   - Monitorar em tempo real
